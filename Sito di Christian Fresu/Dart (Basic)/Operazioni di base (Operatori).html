Operazioni di base (Operatori)

1. Operatori aritmetici
2. Operatori di comparazione
3. Type Test Operator
4. Operatori logici

In dart sono presenti diversi operatori:

  • Operatori aritmetici
  • Operatori di comparazione
  • Type Test Operator
  • Operatori di assegnamento
  • Operatori logici

1. Operatori aritmetici

  • Gli operatori aritmetici sono:

  1. + (Somma)
  1. - (Sottrazione)
  1. * (Moltiplicazione)
  1. / (Divisione)
  1. % (Modulo, resto della divisione di due numeri)
  1. ++ (Incremento di 1)
  1. - - (Decremento di 1)
  1. ~/ (Divisione tra interi)

2. Operatori di comparazione

  • Gli operatori di comparazione sono:

  1. > (Maggiore di)
  1. < (Più piccolo di)
  1. >= (Maggiore o uguale di)
  1. <= (Minore o uguale di)
  1. == (Uguaglianza. Verifica se due valori sono uguali)
  1. != (Disuguaglianza. Verifica se due valori sono diversi)

  • Questi operatori ritornano un booleano come risultato (l’esito del paragone può essere ‘True’ o ‘False’)
  • Questi operatori vengono utilizzati solitamente quando si deve porre qualche condizione

3. Type Test Operator

  • Questi operatori vengono utilizzati per verificare se una variabile contiene un dato di un determinato tipo (ad esempio un numero di telefono non può contenere lettere)
  • Questi operatori sono:

  1. is (Vero, se l’oggetto è di quel tipo specifico)
  1. is! (Falso, se l’oggetto non è di quel tipo specifico)

Esempio 1

void main() {

	int n = 2;
	print(n is int);

}

Esempio 2

void main() {

	double n = 2.20;
	var num = n is! int;
	print(num);

}

4. Operatori logici

  • Gli operatori logici servono per combinare due o più condizioni
  • Questi operatori restituiscono come risultato un booleano
  • Gli operatori logici sono:

  1. && (And) ⇒ Il suo risultato sarà ‘True’, solo se tutte le condizioni vengono rispettate
  1. || (Or) ⇒ Il suo risultato sarà ‘True’, se almeno una delle condizioni viene rispettata
  1. ! (Not) ⇒ Il suo risultato sarà l’opposto dell’esito dell’espressione

Esempio (And)

void main() {

	var a = 10;
	var b = 12;
	var res = (a < b) && (b > 10);
	print(res);

}
