Operazioni (su valori numerici)

  • La libreria di dart permette di effettuare numerose operazioni sui valori numerici

Eccone alcune:

  1. hashcode ⇒ Ritorna un “codice hash” (una sequenza di bit correlata al dato di input) per un valore numerico
  1. isFinite ⇒ ‘True’ se il numero è finito (altrimenti ‘False’)
  1. isNegative ⇒ ‘True’ se il numero è negativo (altrimenti ‘False’)
  1. isEven ⇒ Ritorna ‘True’ se il numero è pari
  1. isOdd ⇒ Ritorna ‘True’ se il numero è dispari

Sintassi (Operazioni sui valori numerici)

variable_name.propertyname

Esempio

void main() {

	int posNum = 10;
	print(posNum.isEven); //Utilizzo 'isEven'

}

Metodi predefiniti (per valori numerici)

  • Esistono alcuni metodi (che operano sui numeri) predefiniti, cioè già presenti nel linguaggio.

Eccone alcuni:

  1. abs() ⇒ Ritorna il valore assoluto di un numero
  1. ceil() ⇒ Ritorna il più piccolo Integer non più piccolo del numero
  1. floor() ⇒ Ritorna il più grande Integer non più piccolo del numero
  1. remainder() ⇒ Ritorna il resto troncato dopo aver diviso due numeri
  1. round() ⇒ Ritorna l’Integer più vicino a quel valore
  1. toDouble() ⇒ Ritorna il Double equivalente a quel numero
  1. toInt() ⇒ Ritorna l’Integer equivalente a quel numero
  1. truncate() ⇒ Ritorna un Integer dopo aver rimosso le cifre decimali

Sintassi (per metodi predefiniti)

variable_name.method();

Esempio 

void main() {

	var a = -2;
	print(a.abs());

}

Operazioni (su stringhe)

  • Per concatenare le stringe viene usato l’operatore ‘ + ‘

Esempio

void main() {

	String s1 = "Hello";
	String s1 = "world";
	res = s1 + s2; //Concatenazione

	print("La concatenazione delle due stringhe: ${res});

}

  • Un altro metodo è quello di utilizzare ‘ ${} ‘

Esempio

void main() {

	int n = 1+1;
	String s1 = "La somma di 1 e 1 è ${n}";

	print(s1);

}

Metodi predefiniti (per stringhe)

Ecco alcuni metodi predefiniti per operare sulle stringhe:

  1. isEmpty ⇒ Ritorna ‘True’ se la stringa è vuota
  1. length ⇒ Ritorna la lunghezza della stringa includendo anche gli spazi, i tab e i caratteri a capo

Esempio

void main() {

	String s1 = "Programming Hub";/t

	print("La lunghezza della stringa è: ${str.length}");

}

  1. toLowerCase() ⇒ Converte tutti i caratteri della stringa in minuscolo
  1. toUpperCase() ⇒ Converte tutti i caratteri della stringa in maiuscolo
  1. trim() ⇒ Ritorna la stringa senza spazi vuoti iniziali e finali
  1. split() ⇒ Divide la stringa nel numero di parti specificato e ritorna una lista di sottostringhe

Esempio (split())

void main() {
  var input = "Frase di esempio";

  print(input.split(" "));
