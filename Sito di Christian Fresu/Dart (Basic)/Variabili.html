Variabili

Tipi di variabili
1. Numeri
2. Stringhe
3. Booleani
4. Liste

  • Una variabile è un namespace che tiene in memoria dei valori
  • In dart per dichiarare una variabile si usa la parola chiave ‘ var ‘

Esempio di dichiarazione di una variabile

var name = 'Christian';

  • Tutte le variabili fanno un riferimento all’oggetto che contengono più che al valore. Ad esempio la variabile name dichiarata sopra contiene un riferimento ad un oggetto String che a sua volta contiene il valore ‘Christian’

Tipi di variabili

  • Non tutte le variabili hanno la stessa capienza e non tutti i dati occupano lo stesso spazio. Proprio per questo, esistono diversi tipi di variabili
  • I tipi di variabili ci permettono di definire quanto sarà la capienza di quella variabile

Dart supporta le seguenti tipologie di dati:

  1. Numeri
  1. Stringhe
  1. Booleani
  1. Liste
  1. Mappe

1. Numeri

In Dart i numeri si dividono in due tipi:

  1. Integer ⇒ Valori numerici interi. Ad esempio “10”. Per dichiarare una variabile contenente un Integer si usa la parola chiave ‘ int ‘

  1. Double ⇒ Per i numeri decimali (ad esempio “10.10”). Rappresenta un valore a 64bit (double-precision)

2. Stringhe

  • Le stringhe rappresentano una sequenza di caratteri (ad esempio un nome, un indirizzo, ecc.)
  • Per dichiarare una variabile che deve contenere una stringa, si usa la parola chiave ‘ String ’
  • I valori delle stringhe si possono dichiarare tra apici o doppi-apici

3. Booleani

  • Una variabile di tipo booleano rappresenta i valori ‘True’ o ‘False’
  • Per dichiarare una variabile booleana si deve utilizzare la aprola chiave ‘ bool ‘

4. Liste

  • Un semplice esempio di lista è un array. Un array contiene diversi valori ma dello stesso tipo
  • In Dart, gli array vengono visti come degli “Oggetti Lista”
  • Una Lista è un gruppo ordinato di oggetti
  • Le liste vengono implementate in Dart utilizzando la seguente libreria
dart:core

Ogni Lista ha:

  1. Un identificatore che rappresenta il tipo di dato contenuto nella lista
  1. Un indice (unico) che identifica ogni singolo elemento della lista

Le Liste si possono classificare in:

  1. Fixed Length List (Questo tipo di Lista non può essere cambiato in run-time). Per creare una Lista di questo tipo bisogna seguire queste istruzioni:

    Step 1 ⇒ Dichiarazione della Lista

    var nome_lista = new List(dimensione_iniziale);

    Step 2 ⇒ Inizializzare la Lista

    nome_lista[indice] = valore;

Esempio di Lista 

void main() {

	var lista = new List(3);
	lista[0] = 12;
	lista[1] = 13;
	lista[2] = 11;

	print(lista);

}

  1. Growable List (tipo di Lista che può variare la sua dimensione in run-time). Per creare una lista di questo tipo bisogna eseguire queste istruzioni:

Step 1 ⇒ Dichiarare la Lista 

//Viene creata una lista con dei valori già stabiliti
var nome_lista = [val1, val2, val3]; 

//Viene creata una lista con dimensione = 0
var nome_lista = new List();

Step 2 ⇒ Inizializzazione della lista

nome_lista[indice] = valore;

Esempio di Lista

void main() {

	var num_list = [1,2,3];
	print(num_list);

}
