Array

Dichiarazione

  1. Tipo di dati che conterrà
  1. Parentesi quadre
  1. Nome
  1. Dimensione

int [] numberArray = new int[10];

//Nuovo array che conterrà 10 interi

  • Questa procedura si può fare anche in due step 

int [] numberArray;
numberArray = new int[10];

Accedere agli elementi di un array

  • Con gli array, si può accedere agli elementi salvati in due modi

  1. Accedendo al singolo elemento
  1. Accedendo a multipli elementi

Caso 1 (singolo elemento)

  • Inserire la posizione (indice) dell’elemento nelle parentesi quadre

int num = numberArray[4];

Caso 2 (multipli elementi)

  • Se si vuole sapere quanti elementi sono presenti all’interno di un array bisogna utilizzare il comando ‘.length’

int count = numberArray.length;

  • Per vedere ogni elemento dell’array, come in C, si utilizza un ciclo for che scorre il vettore 

for(int i=0; i<dimensione; i++) {

	system.out.println(numberArray[i]);

}
