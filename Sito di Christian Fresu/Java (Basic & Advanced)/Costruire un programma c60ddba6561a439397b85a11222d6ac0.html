Costruire un programma

Composizione di un programma Java

  • Prima di costruire un programma in Java bisogna avere presente 3 blocchi fondamentali:

  1. Le CLASSI
  1. Il ‘main method’
  1. Il ‘print statement’

Le classi

  • Come una casa; il posto per ogni cosa. Nel nostro caso i membri che eseguiranno le istruzioni saranno dentro alle classi

Esempio di classe

class HelloWorld {

	//Membri
	//Istruzioni

}

Main method

  • Qui vengono inserite ed eseguite tutte le istruzioni che vengono chiamate nelle classi

Esempio di main method

public static void main(String[] args) {

	//Richiamo delle classi e dei metodi

}

Print statement

  • La parte di codice che restituirà a video tutti i risultati del codice
  • Questa parte viene effettuata con l’apposito comando per restituire output

Esempio di programma (a blocchi)

class HelloWorld {

	main_method() {

		System.out.print("Hello World");
	
	}

}
