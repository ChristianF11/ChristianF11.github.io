  • In Java si può leggere il contenuto di altri file o modificarlo attraverso un programma
  • Per fare ciò bisogna utilizzare delle classi dichiarate nel pacchetto ‘java.io’

Creare un file

  • Per creare un nuovo file bisogna usare il metodo 
createNewFile();

  • Il ritorno di questo metodo è un booleano che dice sei il file è stato creato o no

Esempio (creazione file)

import java.io.File;

public class Main {

	public static void main(String[] args) {

		File file = new File("esempio.txt"); //Viene istanziato un nuovo oggetto

		try {

			boolean newFile = file.createNewFile(); //Creazione del file
			System.out.println("Creazione riuscita? " + newFile); //Esito true o false

		}

		catch (Exception e) {

			System.out.println("Ooops! C'è stato un errore"); //Gestione eccezione 

		}

	}

}

Scrivere in un file

  • Per scrivere in un file esistono diverse classi

  1. FileWriter = Per scrivere interi e stringhe
  1. BufferedWriter = Usata se i numeri da scrivere sono molti. Migliora le prestazioni del programma
  1. FileOutputStream = Usata per scrivere direttamente tutti i tipi di dati in un file

  • Per scrivere è necessario usare il metodo
write();

Esempio 

import java.io.FileOutputStream;
import java.io.File;
import java.io.IOException;

pubic class Main {

	public static void main(String args[]) {

		FileOutputStream writeFile = null;
		File file;
		
		String myText = "Scrivo qualcosa nel file";

		try {

			file = new File("esempio.txt"); //Nuovo oggetto di tipo 'File'
			writeFile = new FileOutputStream(file); //Oggetto di tipo 'FileOutputStream'

			byte[] byteData = myText.getBytes(); //Conversione da string a bytes

			writeFile.write(byteData); //Viene scritto il dato convertito in byte (byteData)
			
			System.out.println("Operazione andata a buon fine!");

		}

		catch(IOException ioe) {

			System.out.println("Ooops! C'è stato qualche problema"); //gestione eccezione

		}

	}

}

Leggere un file

  • Si possono usare due classi

  1. FileInputStream
  1. BufferedReader = Per leggere file lunghi (e migliorare l’efficienza)
  1. A seguire, si userà il metodo 
read();

Esempio 

import java.io.FileInputStream;

public class Main {

	public static void main(String[] args) {

		try {

			FileInputStream myFile = new FileInputStream("esempio.txt");
			int i = 0;
			while(i=myFile.read()!=-1) //Viene letto ogni carattere del file
				System.out.println((char)i); //Vengono stampati tutti i caratteri

			myFile.close(); //Viene chiuso il file dopo la lettura

		}

		catch (Exception e) {

			System.out.println(e.getMessage());

		}

	}

}

Eliminare un file

  • Si usa il metodo 
delete(); //Metodo che appartiene alla classe 'File'
  • Questo metodo ritorna un valore booleano che dice se il file è stato eliminato o no

Esempio

import java.io.File;

public class Main {

	public static void main(String[] args) {

		File file = new File("esempio.txt"); //Questo è il file da eliminare
		boolean deleteMe = file.delete(); //Viene usato il metodo per eliminare il file

		System.out.println("File eliminato? " + deleteMe); //Esito operazione

	}

}
