Packages

  • I packages (pacchetti) in Java corrispondono alle librerie di C
  • Un package in Java è un gruppo di classi, interfacce e sub-package simili
  • I packages rendono un codice riutilizzabile ed efficiente
  • Per richiamare un package basta usare la parola chiave ‘import’ più il nome della classe (NON serve riscrivere l’intera classe)

import packageName.subPackageName.ClassName;

  • I packages si dividono in due categorie

  1. Built - in
  1. User - defined

1. Built - in

  • Sono già presenti in Java

Alcuni esempi:

  • ‘java.util’ (Classi relative al tempo, array, scanner, strutture dati)
  • ‘java.io’ (Input/Output operations)
  • ‘java.lang’ (Classi fondamentali per il design degli oggetti)
  • ‘java.awt’ (Contiene gli elementi GUI comuni)

Esempio (di importazione)

import java.ClassName;

Esempio (importare tutto)

import.java.*;

2. User - defined

  • Pacchetti creati dall’utente
  • Per creare un pacchetto bisogna usare la parola chiave ‘package’

Esempio (creazione pacchetto)

package cartoon;

public class Tom {

	//Body

}

  • Per importarla:
import cartoon.Tom;
